5. User rights

        5.1. Permission to access

        In accordance with the legislation in force, the user has a right of access to their personal data.

        This right of access is defined as the possibility for the user to know the personal data that an organization holds about         them.

        The exercise of the right of access allows the user to know whether the data concerning them is processed and how it is         processed. Using this right of access, the user can also check the accuracy of the data.

        5.2. Right to rectification

        The user can request the rectification of inaccurate or incomplete information concerning them.

        The company CLIC'N CLAC, responsible for data processing, undertakes, in accordance with the legislation in force, to         communicate to the other recipients of the data the corrections made - for example to business partners - unless such         communication would require disproportionate efforts.

        5.3. Right to deletion

        The user has a right to request the erasure of personal data concerning them.

        The user may request the deletion of personal data concerning them if it enters one of the following situations:

        - The data is used for prospecting purposes
        - The data are not or no longer necessary with regard to the objectives for which they were initially collected or processed
        - The user withdraws consent to the use of your data
        - The data are subject to unlawful processing (for example, publication of hacked data)
        - The data was collected when the user was a minor as part of the information society
        - The data must be erased to comply with a legal obligation
        - The user has objected to the processing of their data and the person responsible for the file has no legitimate or         compelling reason not to respond to this request

        However, this right to erasure does not apply to the files necessary to fulfill the following objectives:

        - Exercise of the right to freedom of expression and information
        - Enforce a legal obligation or to exercise a public interest mission
        - Reasons of public interest in the field of health
        - Archiving in the public interest, scientific or historical research.
        - Finding, exercising or defending legal claims

        5.4. Right to restriction of data processing

        The user has the right to ask an organization to temporarily freeze the use some of its data.

        If the user disputes the accuracy of the data used by the organization or objects to their data being processed, a legal         period is granted to the organization so that it can proceed to the verification and / or the examination of the application.         During this period, the user has the option of asking the organization to freeze the use of their data.

        Thus, the organization should no longer use this data but should keep it.

        Conversely, the user can use this right to restriction to ask the organization to retain certain data in the event that this         organization wishes to erase this data.

        This may allow the user to keep certain data, for example in order to exercise a right.

        5.5. Right to portability

        In accordance with the legislation in force, the user has, under certain conditions, a right to portability.

        This right allows the user to retrieve the data they have communicated in an open and machine-readable format. The user         can thus easily store or transmit them from one information system to another, with a view to their reuse for personal         purposes.

        5.6. Right of opposition

        The user can object at any time to an organization using some of its data.

        The user must put forward “reasons taking into account their particular situation” except in commercial prospecting cases,         which they can oppose without cause.
